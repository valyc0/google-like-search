Manuale Utente

Benvenuto nel Sistema di Ricerca Multi-Formato

Questa guida ti aiuterà a utilizzare al meglio tutte le funzionalità del sistema.

Come caricare un documento

  1. Accedi all'interfaccia di upload
  2. Seleziona il file dal tuo computer
  3. Clicca su "Carica documento"
  4. Attendi la conferma di caricamento

Come effettuare una ricerca

Inserisci le parole chiave nella barra di ricerca e premi Invio. Il sistema cercherà in tutti i documenti caricati e mostrerà i risultati più rilevanti con le parti di testo che corrispondono alla tua ricerca evidenziate.

Formati supportati

  • Documenti di testo: TXT, DOC, DOCX, ODT, RTF
  • Fogli di calcolo: XLS, XLSX, ODS, CSV
  • Documenti web: HTML, XML
  • Documenti PDF

Suggerimenti per una ricerca efficace

Utilizza termini specifici e rilevanti. Più la tua query è precisa, migliori saranno i risultati. Puoi cercare frasi complete mettendole tra virgolette.

Per assistenza tecnica, contatta il supporto IT.
